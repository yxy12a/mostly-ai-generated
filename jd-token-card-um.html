❓常见问题

Token 额度卡过期了怎么办？ ▼
额度卡通常有 30 天有效期。过期后未使用的额度将自动清零，无法恢复。建议在有效期内合理规划使用，或联系京东云客服申请延期（视具体情况而定）。
如何查询剩余 Token 数量？ ▼
登录京东云控制台，进入"大模型服务" → "额度管理"即可查看实时余额。也可以通过 API 接口查询：GET /v1/quota/remaining。
Token 消耗是如何计算的？ ▼
Token 消耗 = 输入 Token 数 + 输出 Token 数。中文通常 1 个汉字 ≈ 1-2 个 Token，英文 1 个单词 ≈ 1-2 个 Token。具体取决于模型分词规则。系统会在每次 API 调用返回的响应头中包含实际消耗的 Token 数量（X-Consumed-Tokens）。
可以多个项目共享一张额度卡吗？ ▼
可以。额度卡绑定的是京东云账号，账号下的所有应用和项目都可以共享该额度。建议为不同环境（开发/测试/生产）创建子账号并分配相应权限。
